# 实现单词选择判断组件

## 方案选择
选择**点击选择单词**的方案，因为：
- 实现更简单，用户体验更好
- 易于判断选择状态和正确性
- 符合教学场景的交互习惯

## 组件功能
1. **显示句子**：将句子中的每个单词显示为可点击元素
2. **点击选择**：用户点击单词进行选择
3. **错误反馈**：点击错误单词时抖动组件
4. **正确反馈**：
   - 单个正确单词：蓝色高亮
   - 全部正确选择：绿色高亮
5. **状态管理**：跟踪已选择的单词和正确性

## 技术实现
1. **创建新组件**：`docs/.vuepress/theme/components/Question/WordSelector.vue`
2. **组件结构**：
   - 句子显示区域，每个单词为可点击元素
   - 选择状态和正确性的视觉反馈
   - 错误时的抖动动画

3. **核心逻辑**：
   - 解析句子，提取单词
   - 处理单词点击事件
   - 验证选择的正确性
   - 管理选择状态和反馈

4. **使用方式**：
   ```vue
   <WordSelector 
     :sentence="'We love English'"
     :correctWords="['We', 'love']"
   />
   ```

## 样式设计
- 单词显示为可点击的元素
- 正确选择的单词：蓝色高亮
- 全部正确选择：整体绿色高亮
- 错误选择：组件抖动
- 清晰的视觉层次和交互反馈

## 预期效果
学生可以通过点击句子中的单词进行选择，系统会即时反馈选择是否正确，帮助学生掌握句子成分分析能力。
