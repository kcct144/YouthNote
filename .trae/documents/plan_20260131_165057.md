# 实现句子切分练习组件

## 组件功能
- 显示需要切分的句子
- 允许学生点击句子中的空隙处添加分隔符
- 判断点击位置是否正确
- 正确时添加分隔符
- 错误时抖动组件

## 技术实现
1. **创建新组件**：`docs/.vuepress/theme/components/Question/SplitSentence.vue`
2. **组件结构**：
   - 显示句子文本
   - 监听点击事件
   - 计算点击位置
   - 验证切分是否正确
   - 实现正确/错误的反馈动画

3. **核心逻辑**：
   - 将句子分割为可点击的片段
   - 监听点击事件并计算位置
   - 与正确答案比对
   - 实现抖动动画效果
   - 管理切分状态

4. **使用方式**：
   ```vue
   <SplitSentence 
     :sentence="'We love English'" 
     :correctSplitIndex="2"
   />
   ```

## 样式设计
- 句子显示为可点击的文本
- 分隔符使用特殊样式标记
- 错误时的抖动动画
- 正确时的成功反馈

## 预期效果
学生可以通过点击句子中的空隙处尝试切分，系统会即时反馈切分是否正确，帮助学生掌握句子结构分析能力。
