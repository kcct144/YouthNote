# 实现匹配题组件

## 组件功能
- 显示左侧问题列表和右侧答案列表
- 支持点击选择进行匹配
- 验证匹配是否正确
- 提供正确/错误的视觉反馈
- 支持重置功能

## 技术实现
1. **创建新组件**：`docs/.vuepress/theme/components/Question/Matching.vue`
2. **组件结构**：
   - 左侧问题区域
   - 右侧答案区域
   - 匹配状态显示
   - 反馈信息区域

3. **核心逻辑**：
   - 管理用户的匹配选择
   - 验证匹配关系的正确性
   - 提供视觉反馈和动画效果
   - 支持重置功能

4. **使用方式**：
   ```vue
   <Matching 
     :leftItems="['主语', '谓语', '宾语']"
     :rightItems="['We', 'love', 'English']"
     :correctMatches="[[0, 0], [1, 1], [2, 2]]"
   />
   ```

## 样式设计
- 左右分栏布局
- 可点击的选项卡片
- 匹配成功/失败的视觉效果
- 正确匹配时的连线动画
- 错误时的抖动反馈

## 预期效果
学生可以通过点击左侧问题和右侧答案进行匹配，系统会即时验证匹配是否正确，帮助学生掌握知识点之间的对应关系。
