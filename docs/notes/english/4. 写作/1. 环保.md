## 模版句型

### 「建议开头（应该）」+「环保建议（应该）」

| 环保建议（应该）                            | 中文                                   |
| ------------------------------------------- | -------------------------------------- |
| clean up + 地点                             | 打扫...                                |
| protect the animals                         | 保护动物                               |
| use reusable cups (instead of plastic cups) | 使用可重复使用的杯子（而不是塑料杯子） |
| use cloth bags (instead of plastic bags)    | 使用布料袋（而不是塑料袋）             |
| plant more trees                            | 种植更多的树                           |
| pick up the rubbish                         | 捡起垃圾                               |
| make rules                                  | 制定规则                               |
| save water                                  | 节省水                                 |
| save electricity                            | 节省电力                               |

### 「建议开头（不该）」+「环保建议（不该）」

| 环保建议（不该）           | 中文           |
| -------------------------- | -------------- |
| throw rubbish in the river | 把垃圾扔进河里 |
