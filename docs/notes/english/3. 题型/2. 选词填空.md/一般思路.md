### 一、四大词性

| 情况                      | 词性   |
| ------------------------- | ------ |
| 限定词后，形容词后        | 名词   |
| 主语后，情态动词后，to 后 | 动词   |
| 名词前，系动词后          | 形容词 |
| 主语前，有逗号，句子长    | 连词   |

### 二、其他思路

| 情况                         | 思路                  |
| ---------------------------- | --------------------- |
| \_\_\_\_, 开头               | 副词                  |
| 复合不定代词                 | 形容词需要后置        |
| how 后                       | 形容词或 to do        |
| a/an/the 后                  | 名词短语              |
| 介词后                       | 名词、代词、-ing 分词 |
| 动词后                       | 介词、副词、宾格      |
| it's + \_\_\_\_ + to do sth. | 形容词                |
| 什么都不缺 | 副词 |
