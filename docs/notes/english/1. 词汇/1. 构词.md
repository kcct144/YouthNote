## eco-

eco- 表示“经济的、生态的”

- eco：（游戏的）经济局；（空调的）节能模式
- eco-friendly：（环境友好的）
- economy（经济）
- economic（经济的）
- ecosystem（生态系统）
- ecology（生态学）

## re-

re- 表示“重新、再次”

- repeat 重复（再次做同一动作）
- reply 回复（对之前的请求/问题进行回应）
- return 返回（回到原状态/地点）
- review 复习（重新看一遍以巩固）
- realize 意识到（重新思考后想到）
- regret 后悔（重新思考后感觉）
- repair 修复（重新回到正确状态）
- report 报道（将信息再次传递出去）
- record 记录（将信息再次写下来）
- reusable 可重复使用的
- recycle 回收；再利用
