## 非谓语

| 对比 | 谓语动词        | 非谓语动词                   |
| ---- | --------------- | ---------------------------- |
| 形式 | 有时态或助动词  | 仅 to do, -ing, -ed 三种形式 |
| 作用 | 作谓语动词（V） | 作其他成分（S、O、C、M）     |

M：修饰语（包括名词修饰语和动词修饰语）

- 作主语（S）：
    - ==Seeing== is believing.
    - It's good ==to see you==.
- 作表语（C）：Doing is ==learning==.
- 作宾语（O）：I like ==reading==.
- 作修饰语（M）：
    - a ==sleeping== cat
    - a cat ==sleeping in the room==
