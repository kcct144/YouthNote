## 名词修饰语

名词短语：名词 + 名词修饰语（定语）

- 定语
    - 前置
        - 限定词 a tree
        - 形容词 a big tree
        - 名词 an apple tree
    - 后置
        - 介词短语 a tree in the garden
        - 非谓语 a tree growing in the forest
        - 定语从句 a tree that I can climb

Tip：
1. 可数名词一般不以单数形式单独出现
2. 一般都会在前面加限定词，或以复数形式出现
3. 除非这个名词是不可数名词，或构成了一些固定搭配
