# 【语法选择】冠词 a/an/the

::: card
考点：冠词

全称：冠词 a/an/the

关键词：`不定冠词 a/an`、`定冠词 the`、`限定词`
:::

## 一、讲解

冠词，是一种限定词，放在名词前面限定名词的范围和数量。

分为三种：

1. 不定冠词 a/an：泛指，限定数量一个
2. 定冠词 the：特指，但不限定数量
3. 零冠词：泛指，但不限定数量（即不加冠词，一般是复数或不可数名词）

::: tip 什么是泛指/特指？

- 泛指：“一类人”或“一类事物”首次出现时，常用 a/an 泛指，用于引出话题。
- 特指：再次提及某些事物时，就用定冠词 the 来特指（特定的指代）。此时双方都知道是哪个事物。

一般技巧：

1. 能够翻译成“一个”，就用 a/an；能够翻译成“这个；那个”，就用 the
2. there be 和 have 后面一般都用 a/an
3. 由于 a/an 常用来引出话题，因此第一次提及某个事物常用 a/an，再次提及一般用 the

:::
