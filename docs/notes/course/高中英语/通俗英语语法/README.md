## 📚 7天高中通俗英语语法课程

用7天时间，帮孩子搭建清晰的语法框架，告别"一学就会，一用就废"的困境。课程将复杂概念转化为生动比喻，让语法学习变得直观、有趣、高效。

## 💡 课程核心目标

本课程旨在通过7天集中学习，帮助高中生攻克语法难关，实现三个核心目标：

:::: card-grid
::: card
### 🧩 构建语法框架
理解句子成分、词性、从句等核心概念，扫清听课和阅读语法书的障碍，建立系统知识网络。
:::

::: card
### 🔍 掌握句子分析
学会拆解复杂长句，快速识别句子主干与修饰成分，提升阅读理解速度和准确度。
:::
::::

## 📅 7天课程详细安排

| 学习日 | 核心主题 | 学习要点 |
|-------|---------|----------|
| Day 1 | 解密英语句子的DNA | 五大基本句型、句子成分、及物/不及物/系动词 |
| Day 2 | 名词的换装游戏 | 名词短语、前置与后置修饰语（定语从句等） |
| Day 3 | 动词变形记 | 时态、语态、助动词、情态动词、非谓语动词 |
| Day 4 | 句子的嵌套艺术 | 名词性从句、状语从句、定语从句 |
| Day 5 | 给句子贴标签 | 介词短语功能（作定语、状语、表语等） |
| Day 6 | 让句子告别"人格分裂" | 主谓一致三原则、可数/不可数名词、be动词 |
| Day 7 | 大通关：解决实际问题 | 综合复习、句子分析实战、语法填空练习 |

## 🌟 教学亮点

::: card
### 🔄 化繁为简
将抽象的语法术语转化为大白话，并运用生动比喻，降低理解门槛。

### 🏗️ 框架先行
强调知识体系的构建，先搭好"语法房子"的骨架，再填充细节，避免知识碎片化。

### ✍️ 讲练结合
每节课都配有针对性练习，从理解到应用，确保孩子真正掌握并能解决实际问题。
:::

## 👨‍🏫 给家长的话

语法是英语学习的"交通规则"。很多孩子成绩卡壳，不是因为不努力，而是因为规则摸不清，导致课堂和作业效果都不好。

这7天课程，就是给孩子一张清晰的"语法地图"和一套实用的"分析工具"。我们不止教知识，更教方法，目标是让孩子离开课堂后，有能力自己读懂规则、分析句子、解决问题。
