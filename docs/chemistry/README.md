# 什么是化学？

化学是分子层次上==研究物质组成、结构、性质、转化及应用的科学==。

## 化学研究的对象 ⭐

化学研究的对象是==物质==。化学不仅研究自然界已经存在的物质（如氧、铁、水），还创造和研究自然界原本不存在的物质（合成药物、新材料）。
