## 一、模仿朗读

容易出现的问题
- 单词读错
    - 复数 -s / -ed
    - -ed
    - 单词结尾辅音 t, d, k, g...
- 语速过快
- 节奏混乱，停顿位置不当
- 内容不完整

推荐方法：
1. 播放录音前：观察思考
    - 不认识的单词：怎么读
    - 比较长的句子：怎么停顿
2. 播放录音时：跟着默读
    - 自己会读错、停顿错的地方
3. 准备时间
    - 优先练习长句和生词多的句子
    - 把标记的生词单独练几遍，确认发音
    - 时间充裕可以完整读一遍，磕绊的地方多读一下
4. 朗读时间
    - 注意语速和节奏
    - 确保单词读到位
    - 以短语为单位稍微停顿
    - 读错不要重读，不要有“呃”
    - 虚词弱读

## 二、信息转述

推荐方法：
1. 播放前：预测
    - 主要话题
    - 可能出现的单词
    - 可能用到的句型
2. 第一次播放：记录最关键的动词/名词/形容词
    - 实在记不了就写开头，待会再补齐
3. 第二次播放：补齐信息，确定句型
    - 如果关键信息没有听完，就先听关键词；
    - 如果写完关键信息，听听原文用什么句型表达
4. 准备时间
    - 如果不知道用什么句型，就自己造
    - be, like, have 等简单句型
    - 开口复述一遍：检查卡住的地方，换成更简单的说法
5. 转述时间
    - 要点优先：宁可语法有小错，也要把每个要点都说出来
    - 卡住就跳过：某要点说不下去，立刻跳到下一个

::: warning

要点齐全比语法完美更重要！丢失一个要点的扣分往往比一个语法错误更严重。

:::
