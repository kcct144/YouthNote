## 一、时态

时态：时间+状态

- 时间：现在、过去、将来
- 状态：一般、进行、完成

英语表达时态的方式：改谓语动词

- 加助动词（be, have）
- 变化词形（-s, -ing, -ed）

初中常考的时态：
| | 现在 | 过去 | 将来 |
| --- | --- | --- | --- |
| 一般 | 一般现在时⭐️ | 一般过去时⭐️ | 一般将来时⭐️ |
| 进行 | 现在进行时⭐️ | 过去进行时 | 将来进行时 |
| 完成 | 现在完成时⭐️ | 过去完成时 | 将来进行时 ↓ |

判断时态的一般思路：

1. 看标志词（一般是时间状语）
2. 看周围其他谓语动词的时态

时态示例：
| 时态 | 例词 | 含义 |
| --- | --- | --- |
| 一般现在时 | talk/talks | 说 |
| 一般过去时 | talked | 说了 |
| 一般将来时 | will talk | 将要说 |
| 现在进行时 | be talking | 正在说 |
| 现在完成时 | have talked | 已经说了 |
| 过去进行时 | was/were talking | 当时正在说 |

## 二、各时态动词形式

动词各个时态的形式如下：

| 时态 | 现在      | 过去           | 将来           |
| ---- | --------- | -------------- | -------------- |
| 一般 | do/does   | did            | will do        |
| 进行 | be doing  | was/were doing | will be doing  |
| 完成 | have done | had done       | will have done |

需要详细注意的是 do, does, did, done 这四种形式：

| 代表 | 含义     | 形式   |
| ---- | -------- | ------ |
| do   | 动词原形 |        |
| does | 三单     | 加 -s  |
| did  | 过去式   | 加 -ed |
| done | 过去分词 | 加 -ed |

### 1. 三单形式

一般来说，三单形式加 -s 遵循以下规律：

| 情况                 | 规律             |
| -------------------- | ---------------- |
| 一般情况             | 加 -s            |
| s, x, ch, sh, o 结尾 | 加 -es           |
| 辅音字母 + y 结尾    | 改 y 为 i 加 -es |

不规则情况：

- be → is
- have → has
- go → goes

### 2. 过去式和过去分词

一般来说，过去式加 -ed 遵循以下规律：

| 口诀 | 情况            | 规律                         |
| ---- | --------------- | ---------------------------- |
| 直   | 一般情况        | 直接加 -ed                   |
| 去   | 不发音的 e 结尾 | 去 e 加 -ed                  |
| 双   | 辅元辅结尾      | 双写最后一个辅音字母再加 -ed |
| 改   | 辅音字母加y结尾 | 改 y 为 i 加 -ed             |

此外，过去式和过去分词有相当多不规则形式。详见“不规则动词“表。

## 三、现在时态

- 一般现在：
  - I play basketball everyday.
  - 我每天打篮球。
- 现在进行：
  - I am playing basketball now.
  - 我现在正在打篮球。
- 现在完成：
  - I have played basketball already.
  - 我已经打过篮球了。

### 标志词

1. 一般现在：频度副词
   - always, usually, often, sometimes, seldom, hardly, never
   - every day, every week, every month, every year
   - once a week, twice a month, three times a year
2. 现在进行：now, Listen! Look!
3. 现在完成：表示完成或持续
   - already, just, yet, ever, never, before
   - so far, in the last few days, for+时间段

## 四、过去时态

- 一般过去：I played basketball yesterday.
- 过去进行：I was playing basketball at 10:00 yesterday.

### 标志词

1. 一般过去：表示过去时间的词
   - ago, yesterday, just now, in the past
   - last week/month/year
   - in 2020
2. 过去进行：具体时刻+过去标志
   - at 10:00 yesterday
   - at that time
   - when 或 while 引导的从句

## 五、将来时态

- 一般将来：I will play basketball tomorrow.
- 将来进行：I will be playing basketball at 10:00 tomorrow.

### 标志词

一般将来：表示将来时间的词

- tomorrow, next week, in the future
- next day/week/month/year/Monday/Tuesday
- in 2050

## 六、选择正确的时态

基本策略：

1. 判断选项时态
2. 看标志词（一般是时间状语）
3. 看周围其他谓语动词的时态

技巧：实际做题中，判断时态不必太精确（精确到一般、进行、完成、现在、过去、将来），如：
- yesterday → 一般过去× | 过去 √
- since → 现在完成 × | 完 √

通用标志词技巧：
- 过去标志：表示过去时间的词（ago, before, yesterday, last week）
- 将来标志：表示将来时间的词（tomorrow, next week）
- 一般标志：频度副词（如 never, always, once a week）
- 进行标志：具体时间节点
- 完成标志：yet, already, since, for+时间段

### 例题

1. ==Since== last year, our school \_\_\_\_\_ free breakfast to students.
    - A. offer（一）
    - B. offered（过）
    - C. has offered（完）
    - D. was offering（过进）
    - 标志词：since（完成时标志），故选 C。
2. We ==played== together happily. And my friends \_\_\_\_\_ playing with him.
    - A. enjoy（现）
    - B. enjoyed（过）
    - C. enjoys（现）
    - D. will enjoy（将）
    - 没有标志词。但根据 played 可知时态为过去。故选 B。
