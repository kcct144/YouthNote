| 成分 | 定义         | 简单理解 |
| ---- | ------------ | -------- |
| 主语 | 句子的话题   | 什么/谁  |
| 谓语 | 对主语的描述 | 怎么了   |

通常来说，主语之外都是谓语，例如：
| 主语    | 谓语            |
| ------- | --------------- |
| The sun | gives us light. |

谓语动词：引出谓语的动词短语
- 位置：主语后
- 形式：有时态

| 简称 | 基本句型                    | 动词类型       |
| ---- | --------------------------- | -------------- |
| SV   | 主语+不及物动词             | 不及物动词     |
| SVO  | 主语+及物动词+宾语          | 及物动词       |
| SVC  | 主语+系动词+表语            | 系动词         |
| SVOC | 主语+及物动词+宾语+宾语补语 | 不完全及物动词 |
| SVoO | 主语+及物动词+宾语+宾语     | 双及物动词     |
