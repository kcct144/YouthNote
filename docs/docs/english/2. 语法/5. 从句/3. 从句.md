## 从句

从句的主要考点：

- 语法选择：连词/引导词的选择，从句的时态
- 选词填空：主语前面的空常用连词或引导词

常见的连词：

- and, but, or
- so because
- while, when, since, until
- before, after

常见的引导词：

- what, who, why, when, where, how
- that, which, if, whether
- whatever, whoever, wherever, whenever

连词搭配：
| 连词搭配 | 释义 |
| --- | --- |
| both ... and | ... 和 ... 都 |
| not only ... but also | 不仅 ... 而且 ... |
| neither ... nor | 既不 ... 也不 ... |
| either ... or | 要么 ... 要么 ... |
| so ... that | 如此 ... 以至于 ... |
| such ... that | 例如 ... |
