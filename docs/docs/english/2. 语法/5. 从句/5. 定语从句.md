## 定语从句

> 定语从句修饰**名词**，放在被修饰的名词后面。
>
> **中考频率**：⭐⭐⭐ 高频。that/which/who 的选择是语法选择常考点，"只用 that 的六种情况"是高频陷阱。阅读理解中识别定语从句有助于理清修饰关系。

### 基本结构

```
先行词 + 关系词 + 从句（陈述语序）
the book  that     I bought yesterday
```

### 关系词分类

| 关系词 | 指代 | 在从句中充当 | 示例 |
|--------|------|------------|------|
| **that** | 人/物 | 主语/宾语 | the book **that** I read |
| **which** | 物 | 主语/宾语 | the book **which** is on the desk |
| **who** | 人 | 主语 | the boy **who** is standing there |
| **whom** | 人 | 宾语 | the boy **whom** I met |
| **whose** | 人/物 | 定语（……的） | the boy **whose** father is a doctor |
| **when** | 时间 | 状语 | the day **when** we met |
| **where** | 地点 | 状语 | the place **where** I grew up |
| **why** | 原因 | 状语 | the reason **why** I left |

### 高频考点：只用 that 的六种情况

| 情况 | 示例 |
|------|------|
| 先行词被**最高级**修饰 | the best book **that** I've read |
| 先行词被**序数词**修饰 | the first thing **that** I did |
| 先行词是 **all, everything, nothing** 等 | all **that** I know |
| 先行词被 **the only, the very** 修饰 | the only person **that** can help |
| 先行词**既有人又有物** | the man and the dog **that** ... |
| 主句以 **who/which** 开头 | Who is the man **that** is talking? |

### 关系词可省略的情况

当关系词在从句中作**宾语**时可以省略：

- the book (**that**) I bought ✅
- the boy (**who/whom**) I met ✅
- the book **that** is on the desk ❌（作主语，不能省）

### 考点速查

| 考试题型 | 常见考点 |
|----------|---------|
| 语法选择 | that vs which；关系词选择；只用 that 的情况 |
| 选词填空 | 名词后可能需要关系词 |
| 阅读理解 | 识别定语从句→理解修饰关系 |
| 写作 | who/which/that 合并简单句→复合句 |
