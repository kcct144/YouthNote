## 时间介词

| 介词 | 场景 | 示例                    |
| ---- | ---- | ----------------------- |
| in   | 年   | in 2026                 |
| in   | 季   | in summer               |
| in   | 月   | in July                 |
| on   | 日   | on Sunday, on March 1st |
| at   | 点   | at 10:00                |

### 特殊情况

- in the morning/afternoon/evening
- at noon/night
- 具体某一天的上午/下午/晚上用 on
  - on Saturday morning

## 其他时间介词

| 介词            | 释义                    |
| --------------- | ----------------------- |
| for + 时间段    | 持续...                 |
| in + 时间段     | 一段时间内              |
| since + 时间点  | 自...以来               |
| from ... to ... | 从...到...              |
| during          | 在...期间               |
| by/until/till   | 到...为止；在...之前    |
| before          | 在...之前               |
| after           | 在...之后               |
| through         | 一直...（从开始到结束） |
