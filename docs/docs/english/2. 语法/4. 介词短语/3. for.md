## for 的用法

1. 为；给（表目的）
    - study for a test 为考试而学习
    - wait for ... 等待...（为...而等待）
2. 因为；由于（表原因）
    - be famous for 因...而闻名
    - thanks for 因...而感谢
3. 对...来说
    - for me 对我来说
    - It's + 形容词 + for sb. + to do sth. 做某事对某人来说是...的
    - be good for 对...有好处
4. 持续...（for + 时间段，完成时标志）
    - for a long time 很长时间
    - study for 2 hours 学了2个小时
