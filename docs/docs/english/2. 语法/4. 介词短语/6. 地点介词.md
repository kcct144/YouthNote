## 地点介词

- 表示“在”的词：
  - ==in + 大地点==：in China, in Meizhou
  - ==at + 小地点==：at the park

| 介词                  | 释义               |
| --------------------- | ------------------ |
| in / inside           | 在...里面          |
| outside               | 在...外面          |
| on                    | 在...上面          |
| under                 | 在...下面          |
| over / above          | 在...上方          |
| in front of           | 在...前面          |
| by / beside / next to | 在...旁边          |
| near                  | 在...附近          |
| from                  | 来自；从...开始    |
| between ... (and ...) | 在...（和...）之间 |
| through               | 穿过               |
| across                | 横过               |
