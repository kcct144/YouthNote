# 选词填空 · 动词与非谓语（专题 11–20）

> 从真题短文取空，每题 4 个选项（从该短文词框随机抽取）。
>
> - ==A 型（情况→词性）==：句子挖空 → 选正确单词
> - ==B 型（词→情况）==：给出单词 → 选它能填入的句子
>
> 💡 ==用法==：先看专题前的线索提示，再做题。每题根据语法线索判断答案的词性/意义。
