# 含other的词

| 含other的词 | 限    | 其他的             |
| ----------- | ----- | ------------------ |
| other       | 限    | 其他的             |
| another     | 限/代 | 另一个             |
| others      | 代    | 其他人             |
| the other   | 限/代 | 另外一个           |
| the others  | 代    | 剩下的人；剩余部分 |
| others'     | 限    | 其他人的           |


| 单词     | 意思               | 后面接       |
| -------- | ------------------ | ------------ |
| few      | 强调少（几乎没有） | 可数名词复数 |
| little   | 强调少（几乎没有） | 不可数名词   |
| a few    | 强调有（有一些）   | 可数名词复数 |
| a little | 强调有（有一些）   | 不可数名词   |


# 感叹句
