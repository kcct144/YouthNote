## 含other的词

| 含other的词 | 限    | 其他的             |
| ----------- | ----- | ------------------ |
| other       | 限    | 其他的             |
| another     | 限/代 | 另一个             |
| others      | 代    | 其他人             |
| the other   | 限/代 | 另外一个           |
| the others  | 代    | 剩下的人；剩余部分 |
| others'     | 限    | 其他人的           |

| 单词     | 意思               | 后面接       |
| -------- | ------------------ | ------------ |
| few      | 强调少（几乎没有） | 可数名词复数 |
| little   | 强调少（几乎没有） | 不可数名词   |
| a few    | 强调有（有一些）   | 可数名词复数 |
| a little | 强调有（有一些）   | 不可数名词   |

## 感叹句

- How + 形 （+主谓）
- What + 名短 （+主谓）

## 百、千、百万、十亿

| 单词     | 意思 | 数            |
| -------- | ---- | ------------- |
| hundred  | 百   | 100           |
| thousand | 千   | 1,000         |
| million  | 百万 | 1,000,000     |
| billion  | 十亿 | 1,000,000,000 |

用法：
1. 数词 + hundred （不加s，几百）
2. hundreds of （加s，数百个）
