## 冠词 a/an/the
- 基本思路：
    1. a/an :泛指任意一个  首次出现  表示“一个”
    2. the ：确定特指  再次提及  表示“这个”

- 常考
    1. have 或 there be 后用 a/an
    2. the + 最高级/序数词
    3. hour, honor, honest 的h不发音，前用an
    4. unit, uniform, university, useful, useless 用a/one
    5. 量词短语 a pair of, a set of…
    6. play+球类，play the+乐器
    7. 不可数名词不能用a/an



- 搭配：the same
- 泛指：有个人（a man）  一只狗（a dog）
- 特指：那个人（that/the man）  那个狗（that/the dog）
