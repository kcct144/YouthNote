## 比较

| 原级     | 比较级        | 最高级        |
| -------- | ------------- | ------------- |
| smart    | smarter       | smartest      |
| outgoing | more outgoing | most outgoing |

1. 一般标志：
   - 比较级有than——he is taller than me.
   - 最高级有the——he is the tallest.
  
:::tip
注意还有个 The + 比较级， the + 比较级 的搭配
:::

2. 特殊变化
 
| 原级       | 比较级 | 最高级   |
| ---------- | ------ | -------- |
| good/well  | better | the best |
| bad        | worse  | worst    |
| many/ much | more   | most     |
| little     | less   | least    |

3. 搭配
    - as+原级+as…——和…一样多
    - much+比较级——…得多
    - the+比较级…，the+比较级…——越…越…
    - one of+最高级——最…的之一
   
   - 形物代可以代替the，加最高级——my best friend
   - the+序数词+最高级——第几…的
   - and并列前后一致

:::tip 
没有明显标记，考虑原级
:::
