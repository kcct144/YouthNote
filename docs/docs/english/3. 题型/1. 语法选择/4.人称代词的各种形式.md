## 人称代词的各种形式

1. 形物代+名词/形+名
   - her book/her iinteresting book
2. 主格：开头/动词前
3. 宾格：
   - 动词后 help me
   - 介词前 for me
4. 反身代词：
   - 主语和宾语是同一个人 I trust myself.
   
| 搭配             |        翻译        |
| :--------------- | :----------------: |
| enjoy oneself    |      玩得开心      |
| teach oneself    |        自学        |
| hurt oneself     |        受伤        |
| by oneself       |        独自        |
| say to oneself   | 自言自语，对自己说 |
| think to oneself |       心里想       |
