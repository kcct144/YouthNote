## 非谓语动词

1. do（动词原形）
    -  情态动词+动原
    -  祈使句开头
2. -ing 分词（现在分词、动名词）
    -  动词作主语
    -  介词+名、代、-ing 分词
    -  -ing 结尾的形容词修饰物
3. -ed 分词（过去分词）
    -  -ed 结尾的形容词修饰人
4. to do（不定式）
    -  ==表目的== （为了做某事）
    -  疑问句 + to do
    -  复合不定代词 + to do
    -  It's + 形 + to do


- 基本思路：
    1. 看搭配
    2. 看一般用法
