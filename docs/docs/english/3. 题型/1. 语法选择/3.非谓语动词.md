## 非谓语动词

- do（动词原形）
    1. 情态动词+动原
    2. 祈使句开头


- -ing分词（现在分词、动名词）
    1. 动词作主语
    2. 介词+名、代、-ing分词
    3. -ing结尾的形容词修饰物


- -ed分词（过去分词）
    1. -ed结尾的形容词修饰人


- to do（不定式）
    1. ==表目的== （为了做某事）
    2. 疑问句+to do
    3. 复合不定代词+to do
    4. It's+形+to do


- 基本思路：
     1. 看搭配
     2. 看一般用法
