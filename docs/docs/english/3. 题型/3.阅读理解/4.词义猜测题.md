## 词义猜测题

- **考察：缺词情况下，理解上下文并推测词义**
- 问法：
    1. What's the meaning of the underlined word?
    2. The underlined word in paragraph …means?

- 方法
    1. 此句后面是对该词的解释
        - XXX is …
        - XXX means …
        - XXX —— XXX
        - XXX who/which …
        - XXX : …
        - XXX，XXX …
    2. 前后有例子（for example,such as,like……）
    3. 因果关系、用途、方式
        - We can share the **route** with others, so they will know where we are.
        - A. the shoes   B. the steps   C. the way 
    4. 并列、同义、近义、反义
        - happy ↑ and delighted ↑    tired ↓ but fulfilled ↑
    5. 构词法（preheat: pre — 提前、heat 加热）
    6. 直接用选项代替，看是否通顺
