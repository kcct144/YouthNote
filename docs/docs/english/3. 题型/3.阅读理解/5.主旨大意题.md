## 主旨大意题

- **考察：理解+概括 文章/段落的能力**（区分主旨vs事实/观点/细节/例子）
- 问法：
    1. What does the passage/paragraph 3 mainly tell us?
    2. What's the ==main idea== of paragraph 2 ?

- 方法
    1. 找中心句（开头、结尾）
        - 开头：先点名主旨，随后用细节解释
        - 结尾：先讲细节，再归纳要点
        - 可能没有中心句，需自己推敲
    2. 注意
        - 不能以偏概全（只是一个方面、一小点、细节）
        - 不能过于笼统（超出范围）
        - 不能主观臆断
        - 不能放大、缩小、偷换主题
