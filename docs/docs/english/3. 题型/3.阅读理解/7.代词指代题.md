## 代词指代题

- **考察：能通过代词，始终跟上作者”正在聊什么“**（保持思路的连贯）
- 问法：
    1. The underlined word ==it/them/they…== refers to…
    2. What does the underlined word ==it== refer to?

- 方法
    1. 通常代词都指代前面的东西（而且一般就在上一句）
    2. 确保数量、人称一致
        - they → 复数名词或多个东西
        - it/this → 单数/句子/动词短语
    3. 可以把选项放到代词位置，看是否合适
