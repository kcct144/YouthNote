## 推理判断题

- **考察：从文中获取具体信息的能力**
- 问法
    1. What can we infer/learn from …?
    2. Which of the following statements …?
    3. What's the writer's opinion/attitude …?
    4. Where do you probably read the passage …?

- 分类
    1. 针对细节
        - 与原文一致/同义（√）
        - 与全文/段落主旨相符（√）
    2. 观点、态度、情感
        - 文中明显表示情感色彩的词
        - 文中人物的话语、行为
    3. 出处、读者、用途
        - 文章结构、话题、人物、文体
    4. 后续发展
        - 理清主旨和作者写作思路
        - 看最后暗示下文的句子
