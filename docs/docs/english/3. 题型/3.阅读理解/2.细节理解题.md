## 细节理解题

- **考察：从文中获取具体信息的能力**
- 问法：各种疑问词
  
| 疑问词   | 意思         | 目标             |
| -------- | ------------ | ---------------- |
| What     | 什么         | 东西、事情       |
| Who      | 谁           | 人               |
| When     | 什么时候     | 时间             |
| Where    | 哪里         | 地点             |
| Why      | 为什么       | 原因             |
| How      | 如何、怎样   | 方式，方法，感受 |
| How many | 多少         | 数量             |
| How much | 多少；多少钱 | 数量，钱         |

- 方法
    1. 划关键词（重复少，特征明显）
    2. 定位（可多重复划出和题目相同的词）
    3. 找答案（通常在本句或前后一句，对比疑问词和选项）
