## 基本策略

-  ==**原则：能不读就不读，能少读就少读**== 
- 阅读方法：获取大意（猜主旨）
    1. 非英文信息：图片、数字、文章结构等
    2. 标题、小标题
    3. 开头/结尾段、文章中心距
    4. 每段开头/结尾句、段落中心句
    5. 转折词（but），总结词后才是重点（中心句）
    6. 例子可以 ==跳过== （for example）
    7. 留意实词（名、动、形、副）不纠结虚词（介、限、助…）
    8. 留意重复出现的词
    9. 常识很重要
   
- 阅读方法：快读定位（猜位置）
    1. 特征明显的词
    2. 重复少的词
    3. 题目顺序和文章顺序差不多
    4. 明确目的，忽略无关内容
    5. 利用已经看出来的文章结构（如时间顺序，观点1/2/3）缩小搜索范围
