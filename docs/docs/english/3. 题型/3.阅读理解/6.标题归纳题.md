## 标题归纳题

- **考察：把握文章核心主旨和高频信息**（从宏观角度提炼标题，不被细节干扰）
- 问法：What's the best title for this passage?

- 方法
    1. 根据全文中心距（参考主旨大意）
    2. 根据重复多的词（一般要在选项中体现）

- 技巧
    1. 选项以偏概全，只是一小部分，排除
    2. 选项太宽泛，排除
    3. 如果我用这个选项做标题，会怎么写？
