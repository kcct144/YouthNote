## 一般思路(词性→情况)

| 词性    | 情况                                       |
| ------- | ------------------------------------------ |
| 名词    | 形容词后、限后                             |
| 动词    | 主语后、情后、 to 后、 be done             |
| 形容词  | 名词前、限后名前、系后（be后）、程度副词后 |
| 副词    | ——，开头、句子开头结尾、动词前后           |
| 连词    | 主语前、并列                               |
| however | ——，开头                                   |
