## 基本思路流程
   
1. 思路
    - 看搭配
    - 看词性
    - 看意思
2. 顺序
    - 先看框，再看题 （词性→情况）
    - 先看题，再看框 （意思→情况）
3. 流程（10min）
    1. 快速看一般思路或搭配（不要纠结，1~3min）
    2. 细看、代入、排除剩余选项
    3. 检查（1~2min）
4. 需要检查：
    - 通读一遍，意思是否通顺
    - 名词：单复数
    - 动词：时态，ing/ed
    - 大小写
