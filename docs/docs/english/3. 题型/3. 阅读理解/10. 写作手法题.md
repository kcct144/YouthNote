## 写作手法题

> 考查：识别文章使用的**修辞手法**和**写作技巧**。
>
> **中考频率**：⭐ 低频但逐年增加。2023-2025 年广东中考每年 1 题，常见于阅读理解最后一问，问文章体裁或写作手法。

### 常见问法

- What writing skill is used in paragraph ...?
- How does the writer develop the idea?
- The writer uses ... to show ...

### 常见写作手法

| 手法 | 英文 | 含义 | 示例 |
|------|------|------|------|
| 举例 | giving examples | 用具体例子解释 | For example, ... / such as ... |
| 列数据 | listing numbers | 用数字增强说服力 | Over 1,000 students ... |
| 对比 | comparison / contrast | 比较相似/不同 | Unlike A, B ... / While A ..., B ... |
| 比喻 | simile / metaphor | 用 A 比作 B | as brave as a lion |
| 拟人 | personification | 把物当人写 | The wind whispered. |
| 引用 | quoting | 引用他人话语 | "..." said Professor Li. |
| 提问 | asking questions | 用问句引出下文 | Have you ever thought ...? |
| 讲故事 | telling a story | 用故事开头吸引读者 | When I was young ... |
| 下定义 | giving a definition | 解释术语含义 | ... means ... |
| 分类 | classifying | 分门别类说明 | There are three types: ... |
| 因果分析 | cause and effect | 分析原因和结果 | As a result, ... / This is because ... |

### 答题思路

1. **看位置**：
   - 开头 → 讲故事 / 提问 / 下定义
   - 中间 → 举例 / 列数据 / 对比
   - 结尾 → 总结 / 呼吁

2. **看标志词**：
   | 手法 | 标志词 |
   |------|--------|
   | 举例 | for example, such as, like |
   | 列数据 | 数字、百分比、年份 |
   | 对比 | but, however, while, unlike, on the other hand |
   | 因果 | because, so, as a result, therefore |
   | 提问 | 问号结尾的句子 |

3. **看文章类型**：
   - 说明文 → 举例/列数据/分类/下定义
   - 议论文 → 对比/因果分析/引用
   - 记叙文 → 讲故事/比喻/拟人

### 考点速查

| 中考考查频率 | 写作手法 |
|-------------|---------|
| ⭐⭐⭐ | 举例、列数据、对比 |
| ⭐⭐ | 提问、讲故事、因果分析 |
| ⭐ | 比喻、拟人、下定义、分类 |
