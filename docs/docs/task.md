## 含 other 的词辨析

:::tabs

@tab 默写

| 单词       | 中文     | 词性     |
| ---------- | -------- | -------- |
| other      | ———————— | ———————— |
| another    | ———————— | ———————— |
| the other  | ———————— | ———————— |
| others     | ———————— | ———————— |
| the others | ———————— | ———————— |

@tab 答案

| 单词       | 中文                   | 词性                      |
| ---------- | ---------------------- | ------------------------- |
| other      | 其他的                 | 限定词（加名词复数）      |
| another    | （三者或以上的）另一个 | 限定词/代词（加名词单数） |
| the other  | （两者的）另外一个     | 限定词/代词（加名词单数） |
| others     | 其他人                 | 代词                      |
| the others | 剩下的人；剩余部分     | 代词                      |

- 限定词：限定名词的词（后接名词）
- 代词：代指名词的词（后面不用接名词）

以上记住中文就行，大概能推测出用法。

:::
