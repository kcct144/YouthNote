| **升华** | 固→气 |   吸    | **樟脑丸消失**、灯丝变细、干冰升华 |
| **凝华** | 气→固 |   放    | **霜、雪、雾凇、冰花、窗花**       |

| 常见填空               | 答案                                   |
| ---------------------- | -------------------------------------- |
| 露的形成属于\_\_\_\_   | **液化**（放热）                       |
| 霜的形成属于\_\_\_\_   | **凝华**（放热）                       |
| 冰化成水属于\_\_\_\_   | **熔化**（吸热）                       |
| 樟脑丸消失属于\_\_\_\_ | **升华**（吸热）                       |
| 雾凇的形成属于\_\_\_\_ | **凝华**（放热）                       |
| 白气的实质是\_\_\_\_   | **小水珠**（液化形成的，不是水蒸气！） |

- **晶体熔化条件**（缺一不可）：①达到熔点 ②持续吸热。
- **沸腾条件**：①达到沸点 ②持续吸热。
- 晶体和非晶体核心区别：**有无固定熔化温度**。

:::

@tab 答案

| 物距 $u$ | 像的性质             | 像距 $v$ | 应用     |
| -------- | -------------------- | -------- | -------- |
| $u>2f$   | **倒立、缩小、实像** | $f<v<2f$ | 照相机   |
| $u=2f$   | **倒立、等大、实像** | $v=2f$   | 测焦距   |
| $f<u<2f$ | **倒立、放大、实像** | $v>2f$   | 投影仪   |
| $u=f$    | **不成像**           | —        | 平行光源 |

| 远视眼 | 太薄，折光太弱 | 视网膜**后** | **凸透镜**（会聚） |

| 平衡力作用在？     | **同一物体**上         | 如：书受到的重力和桌面的支持力       |

| 电动机原理？       | 通电导线在磁场中受力   | 电→动，有电源                        |

| 分类维度           | A 类                                       | B 类                                                       |
| ------------------ | ------------------------------------------ | ---------------------------------------------------------- |
| 可再生 vs 不可再生 | 可再生：太阳能、风能、水能、地热能、潮汐能 | 不可再生：**煤、石油、天然气**（化石能源）、**核能**       |
| 一次 vs 二次       | 一次能源：自然界直接获取                   | 二次能源：加工转化而来（**电能**、汽油、柴油、氢气、酒精） |
| 裂变 vs 聚变       | 裂变：重核**分裂**→较轻核（核电站·可控）   | 聚变：轻核**结合**→较重核（太阳·不可控）                   |
