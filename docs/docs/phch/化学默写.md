[+纯碱是盐]: 纯碱（Na₂CO₃）虽然名字里带"碱"，但从化学组成看，它的阳离子不全是 H⁺，阴离子不全是 OH⁻，所以它属于**盐类**。它的水溶液呈碱性（CO₃²⁻ 水解产生 OH⁻），这是它被称为"纯碱"的原因——但分类上它是盐。

[+氯负一价]: Cl 最外层 7 个电子，容易得到 1 个电子形成 Cl⁻，所以是 -1 价。

[+沉淀不溶于酸]: AgCl 和 BaSO₄ 不仅不溶于水，也不溶于稀 HNO₃。这是因为 HNO₃ 虽然是强酸，但无法与 AgCl 或 BaSO₄ 发生反应（没有气体、水或更难溶的物质生成）。这一特性常用于**离子检验**——加稀 HNO₃ 排除 CO₃²⁻ 干扰后，白色沉淀仍不溶解，才能确认含 Cl⁻ 或 SO₄²⁻。

[+氧化物定义]: 氧化物必须同时满足两个条件：①**只有两种元素**组成；②其中一种必须是**氧元素**。KMnO₄ 有 K、Mn、O 三种元素，不是氧化物。类似的，H₂SO₄、NaOH 也都不是氧化物。

[+有机物例外]: 有机物是**含碳化合物**，但 CO、CO₂、H₂CO₃、碳酸盐（如 CaCO₃、Na₂CO₃）虽然含碳，却归为**无机物**。这是因为它们的结构和性质更接近无机物（如不燃烧、不参与典型有机反应），属于化学史上约定俗成的分类。

[+着火点]: 着火点是物质本身固有的属性，**不能降低或改变**。灭火的原理是"降低温度到着火点以下"或"隔绝氧气"，而不是"降低着火点"。考试中看到"降低着火点"的说法一定是错的。

[+催化剂]: 催化剂的核心是"一变二不变"——**一变**：改变反应速率（可以加快也可以减慢）；**二不变**：反应前后质量和化学性质不变。催化剂**参与反应**（先消耗再生成），只是最终被还原回来；催化剂不能改变**不发生的反应**，也不能改变产物的量。

[+pH试纸]: pH 试纸使用时**不能先用水湿润**。如果湿润了，相当于稀释了待测液——酸性溶液被稀释后 H⁺ 浓度降低，pH **偏大**（往中性靠）；碱性溶液被稀释后 OH⁻ 浓度降低，pH **偏小**（往中性靠）。中性溶液不受影响。
