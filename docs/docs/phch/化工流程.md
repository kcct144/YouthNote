## 概述

化学工艺流程题以真实的工业生产过程为背景，用框图的形式表述生产过程。试题一般由**题头**（题目概述）、**题干**（流程图）、**题尾**（问题）三部分组成。

## 流程图的结构

```
原料 → 预处理 → 核心化学反应 → 分离提纯 → 产品
         ↑                        ↓
      （循环利用物质）        （副产品/废弃物）
```

### 分析流程图的要点

1. **关注"箭头"**：箭头尾部连接反应物，箭头指向生成物（主产物和副产物）
2. **关注"三线"**：
   - 出线和进线：表示物料流向或操作流程
   - 主线：表示主产品
   - 支线：表示副产品
   - 回头线：表示物质循环利用
3. **关注"核心化学反应"**

## 常见考点

### 1. 原料预处理的方法及目的

| 预处理操作 | 目的 |
|-----------|------|
| 粉碎（研磨） | 增大反应物间的接触面积，加快反应速率，使反应更充分 |
| 搅拌 | 使反应物充分接触，反应更充分，加快反应速率 |
| 水浸 | 使可溶物溶解，与不溶物分离 |
| 酸浸 | 溶解金属、金属氧化物、难溶性碱或碳酸盐 |
| 碱浸 | 溶解SiO₂、氧化铝等 |
| 灼烧（煅烧） | 使固体或杂质在高温条件下分解 |

### 2. 流程中反应条件的控制

- **调节pH**：使某反应发生或避免某反应的发生
- **控制温度**：防止某种物质在温度过高时分解或挥发

### 3. 混合物分离提纯的操作

| 操作 | 使用仪器 | 玻璃棒作用 |
|------|---------|-----------|
| **过滤** | 烧杯、漏斗、玻璃棒、铁架台 | 引流 |
| **蒸发** | 酒精灯、蒸发皿、玻璃棒、铁架台 | 搅拌，防止局部温度过高造成液滴飞溅 |
| **降温结晶** | — | 加热浓缩→冷却结晶→过滤→洗涤→干燥 |
| **蒸发结晶** | — | 适用于溶解度受温度影响较小的物质 |

### 4. 判断可循环利用的物质

某设备或流程中的**生成物**同时为另一设备或流程中的**反应物**，则该物质为可循环利用的物质。在流程图中表现为"回头线"或前后均出现同一物质。

### 5. 陌生化学方程式的书写

1. 从流程中寻找反应物和生成物
2. 若从已知信息中找出的反应物和生成物不满足质量守恒定律：
   - 反应在溶液中进行，考虑是否有**水**参与或生成
   - 反应在空气中进行，考虑**空气中各成分**是否参与反应

## 常见的工艺流程

### 1. 海水提镁

海水或卤水 →（加石灰乳）→ Mg(OH)₂沉淀 →（加盐酸）→ MgCl₂溶液 →（蒸发结晶）→ MgCl₂固体 →（电解熔融）→ Mg

主要反应：
- MgCl₂ + Ca(OH)₂ = Mg(OH)₂↓ + CaCl₂
- Mg(OH)₂ + 2HCl = MgCl₂ + 2H₂O
- MgCl₂（熔融）≜ Mg + Cl₂↑（通电）

### 2. 联合制碱法（侯氏制碱法）

向饱和食盐水中通入氨气，形成饱和氨盐水，再通入CO₂，NaHCO₃因溶解度较小而析出，将NaHCO₃加热分解制得Na₂CO₃。

主要反应：
- NH₃ + CO₂ + H₂O + NaCl = NaHCO₃↓ + NH₄Cl
- 2NaHCO₃ ≜ Na₂CO₃ + H₂O + CO₂↑

优点：联合生产NH₄Cl和Na₂CO₃两种产品，提高NaCl利用率，缩短生产流程，减少环境污染。

### 3. 粗盐提纯

**难溶性杂质去除**：溶解 → 过滤 → 蒸发 → 计算产率

**可溶性杂质去除**（含CaCl₂、MgCl₂、Na₂SO₄）：
- 加过量BaCl₂除SO₄²⁻ → 加过量NaOH除Mg²⁺ → 加过量Na₂CO₃除Ca²⁺和过量Ba²⁺ → 过滤 → 加适量盐酸除过量NaOH和Na₂CO₃

> 注意事项
> 1. Na₂CO₃溶液必须在BaCl₂溶液之后加入（过量的BaCl₂由Na₂CO₃除去）
> 2. 盐酸必须在过滤之后加入（否则可能溶解已生成的沉淀）
> 3. 过滤后加过量盐酸，蒸发结晶时盐酸会挥发

## 常见考点

1. 操作名称判断（过滤、蒸发、降温结晶等）
2. 玻璃棒在不同操作中的作用
3. 加入试剂的目的和顺序
4. 循环利用物质的判断
5. 陌生化学方程式的书写
6. 物质成分分析（过量物质、剩余物质）
